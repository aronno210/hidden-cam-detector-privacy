# Privacy Policy — Hidden Cam Detector

**Last updated: April 13, 2026**

## Information We Collect
This app uses Camera, Location, Bluetooth, and WiFi 
permissions solely to detect hidden cameras. 
We do NOT store or share any personal data.

## Third-Party Services
This app uses **Unity Ads** for advertising.
Unity Ads may collect device info for ad purposes.
See: https://unity.com/legal/privacy-policy

## Permissions Used
- Camera: For lens reflection detection
- Location: For WiFi/Bluetooth scanning
- Bluetooth: To detect Bluetooth-enabled hidden cameras
- WiFi: To scan suspicious network devices
